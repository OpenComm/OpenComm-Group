There isone source file which has the name of the Jabber account user stored with its password.

The program is a cmd line application that does the following:

1. Logins into jabber.org
2. Prints the user's buddy list
3. creates a chatroom
4. asks which buddies should be added (Type the names that were printed from the buddy list)
	type end to stop inviting people to chatroom
5. Send and receive messages in the MUC
	type end to exit program
